## 🌟 Overview

Welcome to my personal 3D Developer Portfolio repository! This project is a highly optimized, interactive, and modern web application built natively with HTML, CSS, and JavaScript. 

The application focuses on delivering a premium, recruiter-grade user experience through advanced interactive UI design, glassmorphism aesthetics, dynamic particle engines, and optimized 3D physics rendering. It serves as the central hub for my software engineering projects, technical skills, certifications, and work experience.

## 🚀 Key Features & Architectural Highlights

* **Advanced 3D Rendering & Physics:** Integrated `Three.js` and `VanillaTilt.js` to provide interactive hover physics, cosmic background particle generation, and dynamic geometric structures.
* **Premium Glassmorphism UI:** Stunning transparent layouts featuring responsive light/dark mode toggles with distinct color palettes, backdrop filters, and neon-glow accents.
* **High-Performance Lazy Loading (PDFs):** Features a highly optimized PDF Certificate engine utilizing `PDF.js` combined with an `IntersectionObserver` memory-management algorithm, ensuring zero browser lag while rendering high-res documents.
* **Interactive Animations:** `GSAP`-powered scroll reveals, custom canvas-based particle engines, and fluid water-ripple physics for the contact section.
* **Responsive Sidebar Architecture:** Fully optimized for seamless scaling across desktops, tablets, and mobile devices featuring a glassmorphism slide-out hamburger menu.
* **SEO Optimized & Minified:** Fully compliant semantic HTML structure, dynamic `sitemap.xml` generation, Google Search Console integration, and automated asset minification for blazing fast load times.

## ⚙️ Running Locally

Want to run this project on your local machine to explore the code? It's as simple as running a local web server!

```bash
# 1. Clone the repository
git clone https://github.com/sonararadhya/aradhya-portfolio.git

# 2. Navigate to the directory
cd aradhya-portfolio

# 3. Serve the directory using Python (or any static server)
python3 -m http.server 8000
```
*Then open `http://localhost:8000` in your web browser!*

*(Note: The codebase uses a custom `minify.sh` script to compile raw `styles.css` into `styles.min.css` for production).*
