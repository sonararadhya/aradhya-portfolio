## 🌟 Overview
Welcome to my personal developer portfolio repository! This project is a highly optimized, interactive, and modern web application built natively with HTML, CSS, and JavaScript. 

The application focuses on delivering a premium, recruiter-grade user experience through advanced interactive UI design, glassmorphism aesthetics, dynamic particle engines, and optimized 3D physics rendering. 

## 🚀 Key Features

* **Advanced 3D Rendering:** Integrated Three.js and VanillaTilt to provide interactive hover physics and dynamic geometric structures.
* **Premium Glassmorphism UI:** Stunning transparent layouts featuring responsive light/dark mode toggles with distinct color palettes.
* **High-Performance Lazy Loading:** Features a highly optimized PDF Certificate engine utilizing `PDF.js` with `IntersectionObserver` memory-management, ensuring zero browser lag.
* **Interactive Animations:** GSAP-powered scroll reveals, custom particle background engines, and fluid water-ripple physics for the contact section.
* **Responsive Architecture:** Fully optimized for seamless scaling across desktops, tablets, and mobile devices without losing visual fidelity.
* **SEO Optimized:** Fully compliant semantic HTML structure with dynamic sitemap generation and optimal Google Search Console integration.

## ⚙️ Running Locally

Want to run this project on your local machine? It's as simple as running a local web server!

```bash
# 1. Clone the repository
git clone https://github.com/sonararadhya/aradhya-portfolio.git

# 2. Navigate to the directory
cd aradhya-portfolio

# 3. Serve the directory using Python (or any static server)
python3 -m http.server 8000
```
*Then open `http://localhost:8000` in your web browser!*
